# 变更日志

## [0.0.3] - 2025-06-19

### 新增

- 添加Pug文件格式化功能
  - 支持通过命令面板调用格式化
  - 支持使用 VS Code 标准格式化快捷键（Shift+Alt+F）
  - 集成到 VS Code 格式化提供程序系统

### 优化

- 重构代码结构，提高可维护性
  - 将格式化和编译功能模块化
  - 统一错误处理机制
  - 优化路径解析逻辑
- 统一配置命名空间
  - 使用 `pug-compile.format.*` 管理格式化配置
  - 使用 `pug-compile.compile.*` 管理编译配置
- 改进配置项说明文档
- 代码质量提升
  - 添加 TypeScript 类型定义
  - 优化错误处理和提示
  - 提升代码可读性
